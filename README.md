# 🌍 Clustering des Pays  
Ce projet explore l’application du **clustering** visant  à identifier des **groupes de pays similaires** afin d'allouer de la meilleure manière les ressources disponibles pour leur venir en aide.

# 🚀 Objectifs du projet  
- 📊 **Analyser les tendances d’attribution des aides internationales**  
- 🔍 **Regrouper les pays selon des caractéristiques communes**  
- 🏆 **Identifier les profils types des bénéficiaires**  

# 🛠 Méthodologie  
- **Prétraitement des données** (nettoyage, normalisation)  
- **Application d’algorithmes de clustering** : k-means, hierarchique, DBSCAN  
- **Visualisation des clusters** à l’aide de graphiques interactifs  
- **Interprétation des résultats** et identification des facteurs clés  

# 📂 Structure du projet  
- `projet.Rmd` : Rapport détaillant l’analyse, les méthodes et les résultats  
- `data/` : Données sources utilisées pour le clustering
